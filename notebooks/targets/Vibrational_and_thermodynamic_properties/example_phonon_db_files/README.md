# Data source
The data provided in here has been generated as part of following publication: [High-throughput density-functional perturbation theory phonons for inorganic materials](http://www.nature.com/articles/sdata201865)

If you use this data please cite the above publication along with its data source that can be found at https://doi.org/10.6084/m9.figshare.c.3938023

Its included here just for demonstrating the conversion and data extraction step followed to obtain target data used in training ML models.
