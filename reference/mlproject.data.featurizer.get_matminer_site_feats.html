get_matminer_site_feats#

mlproject.data.featurizer.get_matminer_site_feats(structures_df, site_featurizers=None, n_jobs=8)[source]#

Get site featurized dataframe using pymatgen structure object .

Uses Matminers MultipleFeaturizer

Parameters:

  • structures_df (DataFrame) – pandas Dataframe object column named “structure”(pymatgen structures in dict format) and “site_index”

  • site_featurizers (list | None) – list of matminer site based featurizes to apply on input structures_df

  • n_jobs (int) – number of parallel jobs to run for featurization

Returns:

A pandas dataframe with structure and composition based features for sites

Return type:

DataFrame
