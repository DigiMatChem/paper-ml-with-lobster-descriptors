Abbreviations
-------------

| Abbreviation      | Description |
|-------------------|-------------|
| **ARFS**          | [All relevant feature selection](https://arfs.readthedocs.io/en/latest/#) |
| **CV**            | Cross-Validation |
| **MODNet**        | [Material Optimal Descriptor Network](https://modnet.readthedocs.io/en/latest/)      |
| **PFI**           | [Permutation Feature Importance](https://scikit-learn.org/stable/modules/permutation_importance.html) |
| **RF**            | Random Forest |
| **SHAP**          | [SHapley Additive exPlanations](https://shap.readthedocs.io/en/latest/) |
| **SR**            | Symbolic regression |
| **SISSO**         | [Sure-Independence Screening and Sparsifying Operator](https://doi.org/10.21105/joss.03960) |
| **Rung**          | Number of times mathematical operators applied to input descriptors by SISSO during model training. Higher the number, more complex and accurate models are generated but quickly get  computationally expensive to train.|
| **nD Descriptor** | Number of descriptors combined linearly by SISSO to predict the target property |
